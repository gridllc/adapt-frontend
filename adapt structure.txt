adapt/
└── frontend/
    ├── public/
    │   └── index.html         ← your HTML + Tailwind CDN (or link to built CSS)
    │
    ├── src/
    │   ├── components/        ← all .tsx UI pieces (Sidebar, Icons, etc.)
    │   ├── pages/             ← full-screen views (WelcomeScreen, ChatTutor…)
    │   ├── services/          ← geminiService.ts, other API wrappers
    │   ├── types.ts           ← shared TS interfaces
    │   ├── metadata.json      ← JSON config/prompts you import in TS
    │   ├── App.tsx            ← root React component
    │   └── index.tsx          ← mounts React into the DOM
    │
    ├── .env.local             ← VITE_ env vars for your endpoints
    ├── package.json           ← lists React, TS, Vite, Tailwind deps
    ├── tsconfig.json          ← TS compiler options (jsx, moduleResolution…)
    └── tailwind.config.js     ← if you’re doing a Tailwind build instead of CDN
